Interactive Shader Course · Advanced Track

Terrain & Landscapes

You can raymarch a sphere — now raymarch a world. This track builds an Iñigo-Quílez-style landscape renderer step by step: heightfield marching, noise terrain, slope/height materials, fog and sun. By the end you have a flythrough vista you can post.

Prerequisites: the core course — especially raymarching (L4), fBM noise (L11), and lighting/fog (L7). Terrain uses a different marching trick than SDFs, but every other tool carries over.

Run over HTTP. From the project root: python3 -m http.server 8777 → open http://localhost:8777/advanced/index.html. Each lab has a ▶ Run button.
